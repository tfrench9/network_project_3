run with: python project3.py

required libraries: pandas, numpy, matplotlib, sklearn

Project Description:
1. Processes CSV files into numeric arrays
2. Splits the arrays by state
3. Normalizes and Preprocesses arrays to be in ideal format for training
4. Runs Multiple Regression Algorithms using Quantitative Supervised learning
    a. Random Forest Regressor
    b. Support Vector Regressor
5. Displays graphical output of resultant regressions, as well as simple data comparison.
